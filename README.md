# Syllabase

## How to run 
- navigate to the `Syllabase` folder
- run `npm run dev`
- the app will show up at http://localhost:5173/ 

## Tasks 
1. Imagine you're a newly hired professor at Cal Poly looking to build out your curriculum for Baking 101. Use Syllabase to find and save a Baking 101 course at Cal Poly.
2. Imagine you're a newly hired professor at Cal Poly looking to clarify the lesson objectives for Baking 101. Use Syllabase to navigate to the course that you saved and ask about the learning objectives. 
3. Imagine you're a newly hired professor at Cal Poly looking to create a syllabus for your own Baking 101 class. Use your saved baking course from the previous task to create a syllabus per your liking. 
4. Imagine you are a professor at Cal Poly who just created a new Human Computer Interaction course. Add this course to the Syllabse database.
