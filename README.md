# Syllabase

## How to run 
- navigate to the `Syllabase` folder
- run `npm run dev`
- the app will show up at http://localhost:5173/

## Tasks 
1. Imagine you're a newly hired professor at Cal Poly looking to build out your curriculum for a Computer Science Human Computer Interaction (HCI) course. Use Syllabase to find and save 2 HCI courses.
2. Imagine you're a newly hired professor at Cal Poly looking to compare the learning objectives of the HCI courses you just saved. Use Syllabase to ask about this comparison.  
3. Imagine you're a newly hired professor at Cal Poly looking to create a syllabus for your own Human Computer Interaction class. Use your saved courses from the previous task to create a syllabus per your liking. 
4. Imagine you are a professor at Cal Poly who just created a new Baking 101 course. Add this course to the Syllabase database.
